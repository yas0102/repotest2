Connection directe à la base de données Postgres avec extension PostGIS

Pour ces requêtes spatiales merci de fetcher la géometrie sous forme de GeoJSON et l'aliaser geom comme : ST_AsGeoJSON(geom) AS geom

Ici essayer des requêtes comme:

SELECT id, ST_AsGeoJSON(geom) AS geom FROM maroc

SELECT id, ST_AsGeoJSON(geom) AS geom FROM communes WHERE mean2018 > 110
